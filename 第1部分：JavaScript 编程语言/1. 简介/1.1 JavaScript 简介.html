现代 JavaScript 教程

第1部分：JavaScript 编程语言 > 1. 简介 > 1.1 JavaScript 简介

JavaScript 最初被创建的目的是"使网页更生动"。这门编程语言创造的程序被称为脚本，它们可以直接写在网页的 HTML 中，在页面加载时自动执行。

浏览器中的 JavaScript

JavaScript 最初是专门为浏览器创建的语言，但现在已经可以在很多其他环境中使用。

独特之处

JavaScript 是世界上唯一一门可以完全集成 HTML/CSS 的编程语言。

现代发展

如今的 JavaScript 不仅可以在浏览器中执行，还可以在服务器或任何配备了 JavaScript 引擎的设备上运行。
