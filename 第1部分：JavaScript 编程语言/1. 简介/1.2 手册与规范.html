ECMA-262 规范

ECMA-262 规范是 JavaScript 语言的官方标准，由 ECMA International 组织维护。它详细定义了 JavaScript 的语法、类型、语句、关键字、保留字、运算符和对象等内容。

这份规范是了解 JavaScript 语言最深层次细节的权威来源，适合想要深入理解语言内部工作原理的开发者。

MDN JavaScript 参考

MDN Web 文档提供了全面的 JavaScript 参考资料，包括内置对象、表达式、语句、函数、运算符等详细说明和示例代码。

与规范相比，MDN 文档更加实用和易于理解，提供了丰富的示例和兼容性信息，是日常开发中查询 JavaScript 用法的理想资源。
