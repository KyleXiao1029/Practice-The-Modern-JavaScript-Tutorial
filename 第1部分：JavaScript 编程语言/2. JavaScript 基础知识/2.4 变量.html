变量

什么是变量？

变量是数据的"命名存储"。在 JavaScript 中创建一个变量，我们需要用到 let 关键字。

在较老的脚本中，你也可能发现另一个关键字 var，而不是 let。

如果将变量想象成一个"数据"的盒子，盒子上有一个唯一的标注盒子名字的贴纸。这样我们能更轻松地掌握"变量"的概念。

let message = "Hello!"; // 变量 message 可以被想象成一个标有 "message" 的盒子，盒子里面的值为 "Hello!"
变量命名规则

JavaScript 的变量命名有两个限制：

  • 变量名称必须仅包含字母、数字、符号 $ 和 _
  • 首字符必须非数字
使用常量

声明一个常数（不变）变量，可以使用 const 而非 let。

一个普遍的做法是将常量用作别名，以便记住那些在执行之前就已知的难以记住的值。 使用大写字母和下划线来命名这些常量。
const COLOR_RED = "#FF0000"; const MAX_WIDTH = 1000;
变量命名最佳实践

一个变量名应该有一个清晰、明确的含义，对其存储的数据进行描述。

变量命名是编程过程中最重要的技能之一。快速地浏览变量的命名就知道代码是一个初学者还是有经验的开发者写的。

练习 1：变量的声明与赋值
// 1. 声明两个变量：admin 和 name // 2. 将值 "John" 赋给 name // 3. 从 name 变量中拷贝其值给 admin // 4. 使用 alert 显示 admin 的值（必须输出 "John"） let admin, name; name = "John"; admin = name;
练习 2：变量命名实践

思考如何为以下数据创建恰当的变量名：

// 1. 使用我们的星球的名字创建一个变量 // 2. 创建一个变量来存储当前浏览者的名字 let ourPlanetName = '地球'; let currentUserName = '张三';
注意变量名的可读性和描述性，避免使用过于简单的名称如 planet 或 user。
