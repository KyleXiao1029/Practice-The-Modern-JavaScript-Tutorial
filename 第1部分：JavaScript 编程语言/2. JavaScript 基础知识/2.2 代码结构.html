JavaScript 代码结构

本章将介绍JavaScript代码的基本结构，包括语句、分号和注释的使用规范。

语句
语句是执行行为（action）的语法结构和命令。
alert('Hello');
alert('World');
分号
当存在换行符（line break）时，在大多数情况下可以省略分号。
// 这些语句用分号分隔
alert('Hello')
alert('World')
在大多数情况下，换行意味着一个分号。但是"大多数情况"并不意味着"总是"！
// 这种情况下需要分号，否则会出错
alert(3 +
1
+ 2);
注释
单行注释以两个正斜杠字符 // 开始。
// 这是单行注释
alert('单行注释示例');
多行注释以一个正斜杠和星号开始 "/*" 并以一个星号和正斜杠结束 "*/"。
/* 这是
多行注释
示例 */
alert('多行注释示例');
不要在 /*...*/ 内嵌套另一个 /*...*/。
/*
/* 嵌套注释 */ // 这会导致错误！
*/
